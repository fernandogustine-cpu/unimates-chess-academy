# Uni-Mates Chess Academy Pro Portal

Static Phase 3 starter portal for GitHub Pages.

## Files
- index.html
- styles.css
- script.js

## Features
- Coach dashboard
- Student manager
- Course builder
- Video lesson library
- Puzzle trainer
- PGN analysis starter
- Tournament manager
- Local browser storage

## Important
This version stores data in the browser only. For real accounts and a shared database, upgrade to Supabase/Firebase.
